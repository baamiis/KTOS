◆ TASK_MAIN_STACK_SIZE

#define TASK_MAIN_STACK_SIZE   512

#include <core/ktos.h>

Default stack depth (in 32-bit words) for the main task.

Increase this value if the task uses deep call chains or large local arrays. Each word is 4 bytes, so 512 words = 2 KB of RAM.

◆ ktos_MSG_TYPE

enum ktos_MSG_TYPE

#include <core/ktos.h>

Built-in message type identifiers delivered by the KTOS scheduler.

Your task function receives one of these as MsgType on every invocation. Values above KTOS_MSG_TYPE_SYSTEM_START are free for application use.

WORD my_task(WORD MsgType, WORD sParam, LONG lParam)
{
switch (MsgType) {
case KTOS_MSG_TYPE_INIT: // first call — hardware init
break;
case KTOS_MSG_TYPE_TIMER: // periodic tick
break;
default: // user-defined message
break;
}
return 100; // sleep 100 ms
}
Enumerator

KTOS_MSG_TYPE_TIMER
