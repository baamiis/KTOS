◆ ktos_InitTask()

struct ktos_TASK* ktos_InitTask ( unsigned short int(*)(unsigned short int MsgType, unsigned short int sParam, long lParam)  Func,
INT  StackSize,
INT  QueueSize,
BYTE  TaskID 
)

#include <core/ktos.h>

Create and register a new task with the KTOS scheduler.

Allocates a task control block and stack, initialises the task's message queue, and inserts the task into the circular scheduling ring. Call this for every task before calling ktos_RunOS().

Parameters
FuncPointer to the task function. Signature:
WORD task(WORD MsgType, WORD sParam, LONG lParam);
Return value is the sleep duration in milliseconds, or MSG_WAIT to sleep until explicitly woken.
StackSizeStack depth in 32-bit words (not bytes). Must be large enough for the deepest call chain, all local variables, and interrupt frame overhead. Start with TASK_MAIN_STACK_SIZE (512 words) and reduce if RAM is tight.
QueueSizeMaximum number of messages the task can queue at once. Messages sent to a full queue are silently dropped. TASK_MAIN_QUEUE_SIZE (3) is a safe default.
TaskIDSingle-byte identifier for debug output (e.g. 'M' for main task, 'S' for sensor task).
Returns
Pointer to the newly created task control block. Pass this to ktos_SendMsg() and ktos_WakeUp().
Note
ktos_Emergency() is called and execution halts if any allocation fails.

Example

struct ktos_TASK *sensor = ktos_InitTask(sensor_task, 256, 4, 'S');
struct ktos_TASK *led = ktos_InitTask(led_task, 128, 2, 'L');
ktos_RunOS();

◆ ktos_RunOS()

void ktos_RunOS ( void  )

#include <core/ktos.h>

Start the KTOS scheduler. Never returns.

Initialises the 1 ms system tick timer and transfers control to the first registered task. All tasks must be created with ktos_InitTask() before calling this function.

Note
If called with no tasks registered, ktos_Emergency() is invoked.
ktos_InitTask(my_task, TASK_MAIN_STACK_SIZE, TASK_MAIN_QUEUE_SIZE, 'M');
ktos_RunOS(); // execution never reaches here

#include <core/ktos.h>

Enqueue a message for delivery to a task.

Thread-safe: interrupts are briefly disabled while writing to the queue. May be called from ISR context.

If the target task's queue is full the message is discarded and false is returned — no assertion or error is raised.

Parameters
TaskTarget task (returned by ktos_InitTask()).
MsgTypeMessage type identifier (built-in or user-defined WORD).
sParam16-bit user-defined parameter.
lParam32-bit user-defined parameter.
Returns
true — message queued successfully.
false — queue full or Task is NULL; message dropped.

Example — notify a task from an ISR

void USART_RX_IRQHandler(void)
{
uint8_t byte = USART->DR;
ktos_SendMsg(g_uart_task, MSG_UART_RX, byte, 0);
}

◆ ktos_Sleep()

int ktos_Sleep ( unsigned short int  Delay,
bool  TaskSwitchPermit 
)

#include <core/ktos.h>

Yield the CPU for a given duration and optionally allow other tasks to run.

The task suspends itself for Delay milliseconds. When the timer expires the scheduler delivers KTOS_MSG_TYPE_TIMER on the next invocation. Call from task context only — never from an ISR.

Equivalent to returning Delay directly from the task function; the difference is that ktos_Sleep() can be invoked deep inside a call stack.

Parameters
DelaySleep duration in milliseconds.
  • 0 — yield immediately (run other tasks once, then resume).
  • 1–65534 — sleep for that many milliseconds.
  • MSG_WAIT — sleep indefinitely until ktos_WakeUp() is called.
TaskSwitchPermitTASK_SWITCH_PERMIT (1) — allow other tasks to run while sleeping. TASK_SWITCH_INHIBIT (0) — freeze scheduler; only this task can resume (requires ISR to call ktos_WakeUp()).
Returns
The WakeUpType value supplied by the ktos_WakeUp() call that woke this task, or 0 if woken by the timer.
Warning
Passing both MSG_WAIT and TASK_SWITCH_INHIBIT halts all other tasks until an ISR calls ktos_WakeUp(). Use with extreme caution.

Example

// Inside a task function — wait 200 ms then continue
ktos_Sleep(200, TASK_SWITCH_PERMIT);
// Execution resumes here after 200 ms

◆ ktos_WakeUp()

void ktos_WakeUp ( struct ktos_TASK *  Task,
INT  WakeUpType 
)

#include <core/ktos.h>

Wake a sleeping task immediately from an ISR or another task.

Sets the task's TimerFlag so the scheduler will dispatch it on the next scheduling cycle. Safe to call from ISR context.

Has no effect if the task is not currently sleeping.

Parameters
TaskTask to wake (returned by ktos_InitTask()).
WakeUpTypeArbitrary value returned to the sleeping task by ktos_Sleep(). Use this to communicate why the task was woken (e.g. which peripheral triggered the event).

Example — wake a task from a button ISR

void EXTI0_IRQHandler(void)
{
ktos_WakeUp(g_ui_task, WAKE_BUTTON_PRESS);
}
// Inside g_ui_task:
int reason = ktos_Sleep(MSG_WAIT, TASK_SWITCH_PERMIT);
if (reason == WAKE_BUTTON_PRESS) { handle_button(); }
