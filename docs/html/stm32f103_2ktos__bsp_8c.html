Supported boards: Blue Pill, Nucleo-F103RB, Maple Mini.

Property Value

__attribute__ ( (naked)  )

Cortex-M3 cooperative context switch.

Load the first task's stack and begin execution. Never returns.

Pushes R4-R11 onto the current stack, stores SP into *p_current_sp_storage, loads next_sp into SP, pops R4-R11 of the new context, and returns (BX LR) into it.

Note
Naked function — no compiler-generated prologue/epilogue.

Sets SP to first_task_sp, restores R4-R11, then pops the hardware exception frame (R0-R3, R12, LR, PC, xPSR) to start the task.

Note
Naked function — no compiler-generated prologue/epilogue.
