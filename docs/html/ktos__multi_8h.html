Detailed Description

Portable primitive types, constants, and utility prototypes used throughout the KTOS kernel and all BSPs.

Include this header wherever KTOS types are needed. It is automatically pulled in by ktos.h and ktos_hal.h, so application code rarely needs to include it directly.
